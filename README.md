## Descrição do Projeto

Esse é um sistema de gerenciamento e exibição de veículos usados. oferecendo:

- Cadastro, edição e exclusão de veículos, marcas, modelos e cores (área administrativa).  
- Visualização de veículos na área pública com detalhes e imagens.  
- Sistema de autenticação para administradores (login, logout e dashboard).  
- Upload e gerenciamento de imagens dos veículos.  

O projeto utiliza Bootstrap 5 para o front-end e MySQL como banco de dados.

---

## Funcionalidades

### Área Pública
- Listagem de veículos.  
- Filtro por marcas e modelos.  
- Visualização de detalhes do veículo, incluindo fotos e descrição.  

### Área Administrativa
- Dashboard com estatísticas rápidas.  
- CRUD completo para Veículos, Marcas, Modelos e Cores.  
- Sistema de login/logout para administradores.  

---

## Tecnologias Utilizadas

- **Backend:** PHP 8.2, Laravel 12  
- **Frontend:** Blade, Bootstrap 5  
- **Banco de Dados:** MySQL  
- **Versionamento:** Git e GitHub
